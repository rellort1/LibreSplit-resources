# Left 4 Dead 2 Autosplitter

- **This relies on Game Instructor being enabled!**
- Individual Level and Main Campaign runs should work fine.

## Features (compared to the L4D2 LiveSplit ASL version)

|                                              	| LibreSplit 	|
|----------------------------------------------	|------------	|
| Load removal                                 	|      ✅     	|
| Chapter split                                	|      ✅     	|
| Split at end of intro cutscenes              	|      ❌     	|
| Split chapters on Scoreboard vs Game Loading 	|      ✅     	|
| Split once                                   	|      ❌     	|
| Cutsceneless start                           	|      ❌     	|
